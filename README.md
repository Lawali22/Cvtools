# CVtools — Générateur de CV professionnel

Créez votre CV professionnel en quelques minutes.  
**20 modèles** dont 10 optimisés ATS. Entièrement gratuit, aucune donnée envoyée.

## Modèles disponibles

### ✦ Modèles Design (10)
Executive · Tech · Creative · Minimal · Corporate · Academic · Medical · Legal · Marketing · Startup

### ✓ Modèles ATS (10)
Clarté · Dakar · Sahel · Diplomate · Ingénieur · Consultant · Académique · Humanitaire · Junior · Cadre Supérieur

## Fonctionnalités

- Photo de profil
- Expériences professionnelles
- Formation
- Compétences & langues
- Centres d'intérêt
- Références professionnelles
- Date & signature
- Export PDF
- 100% local — aucune donnée envoyée

## Installation

```bash
npm install
npm run dev
```

## Déploiement

```bash
npm run build
# Le dossier dist/ est prêt à déployer sur Vercel / Netlify
```
